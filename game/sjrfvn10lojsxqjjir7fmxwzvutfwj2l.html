🎮 How to play?

🚀 Basics

Click on adjacent blocks of the same color to blast them. The bigger the group, the more points you get!

🎯 Levels

Reach the target score for each level. If you run out of moves before reaching it, the game ends.

✨ Bonuses:

  • Bomb (💣): Removes a single selected block.
  • Shuffle (🔄): Rearranges all blocks on the board.
  • Perfect Clear:Clear all blocks for +2,000 points!
