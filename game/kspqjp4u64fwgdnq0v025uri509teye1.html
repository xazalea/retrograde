HOW TO PLAY

GOAL

Merge blocks with the same number to create higher values. Try to reach 2048 and beyond!

CONTROLS

Tap or click on a column to drop the current block. The block will fall and stack on top of existing blocks.

MERGING

When two blocks with the same number touch (horizontally or vertically), they merge into one block with double the value!

TIMER

You have 3 seconds to drop each block. If time runs out, the block drops automatically in a random column!

COMBOS

Chain multiple merges in a row to create combos and earn bonus points!

GEMS

Earn gems with every merge. The bigger the block, the more gems you get!

GAME OVER

The game ends when blocks stack up and reach the danger zone at the top!
